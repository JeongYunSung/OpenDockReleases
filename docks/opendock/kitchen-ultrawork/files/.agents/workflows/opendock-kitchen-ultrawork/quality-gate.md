# Kitchen Ultrawork Quality Gate

1. 요청을 메뉴 추천, recipe, substitution, meal plan, shopping list 중 하나 이상으로 분류합니다.
2. household constraints와 pantry를 확인합니다.
3. 소비기한, 시간, 장비, 재료 공유를 기준으로 후보를 좁힙니다.
4. `KITCHEN_RUN.md`에서 run manifest를 만듭니다.
5. 선택된 결과만 `kitchen/` 아래 작성합니다.
6. 알레르기, 교차오염, 식품 범주별 최소 내부 온도, whole cut 3분 휴지, 보관, 재가열, 단위 변환을 검토합니다.
7. 안전 온도, 영양 또는 건강 claim에는 알려진 공공 보건 기관의 HTTPS URL, 조회일, 적용 범위·한계를 기록합니다.
8. kitchen harness를 실행하고 실패 항목을 수정합니다.
9. 최종 결과에 사용한 재료, 추가 구매, leftovers, 안전상 주의와 검증 결과를 요약합니다.
