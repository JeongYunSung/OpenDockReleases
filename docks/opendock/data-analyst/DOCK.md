# Data Analyst

Metric definitions, SQL review, dashboard briefs, and experiment analysis templates.

## Outcome

AI-assisted analysis starts with definitions, assumptions, and validation instead of untraceable numbers.

## Best For

Data analysts, PMs, founders, and operators who need decision-ready metrics.

## What Gets Installed

- Shared agent instructions: `AGENTS.md`, `CLAUDE.md`, and `GEMINI.md`
- Reusable skills for OpenDock/OMA, Codex, and Claude Code
- A Cursor project rule scoped to this dock
- Role-specific reference files and prompt templates

## First Run

`opendock install opendock/data-analyst@1.0.0`

After installation, open `README.md` in the target project and follow the first three steps.
