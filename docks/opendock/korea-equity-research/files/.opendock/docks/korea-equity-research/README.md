# Korea Equity Research 사용 안내

한국 상장 종목을 기준일, 공시, 가격 흐름과 반대 시나리오로 조사합니다.

## 시작하기

1. 종목명, 단축코드, 시장, 기준일과 궁금한 내용을 알려줍니다.
2. agent에게 리서치를 직접 요청합니다: “삼성전자 005930을 기준일과 공시 출처를 포함해 조사해줘.”
3. 출처와 지표를 따로 정리할 필요가 있으면 `EQUITY_RESEARCH_RUN.md`에서 필요한 section만 골라 사용합니다.

## 검토

평소에는 요청한 리서치를 바로 작성합니다. 사용자가 검토를 요청하면 현재 결과물만 `KOREA_EQUITY_RESEARCH.md` 기준으로 AI가 직접 검토하며, 과거 결과물이나 프로젝트 전체는 검사하지 않습니다.

사용자가 만든 결과물은 사용자 소유이며 Dock 업데이트나 제거 대상이 아닙니다.
