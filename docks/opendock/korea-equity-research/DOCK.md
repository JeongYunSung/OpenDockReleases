# Korea Equity Research

한국 상장 종목을 기준일, 공시, 가격 흐름과 반대 시나리오로 조사합니다.

## 설치 후 생기는 것

- `.opendock/docks/korea-equity-research/README.md`: 빠른 사용 안내
- `.opendock/docks/korea-equity-research/KOREA_EQUITY_RESEARCH.md`: 한국 주식 리서치 기준
- `.opendock/templates/korea-equity-research/EQUITY_RESEARCH_RUN.md`: 선택형 작업 메모
- `.agents/skills/opendock-korea-equity-research/SKILL.md`: agent 작업 절차
- 루트 `AGENTS.md`: 짧은 요청 라우팅과 안전 규칙

## 사용 방법

종목, 시장, 기준일과 질문을 알려주면 공식 출처를 확인해 리서치를 바로 작성합니다. 템플릿은 정리가 필요할 때만 사용하고 현재 작업에 필요한 section만 선택합니다.

> 삼성전자 005930을 기준일과 공시 출처를 포함해 조사해줘.

## 검토 방법

사용자가 검토를 요청하면 현재 결과물만 `KOREA_EQUITY_RESEARCH.md` 기준으로 AI가 직접 검토합니다. 요청하지 않은 과거 결과물이나 프로젝트 전체는 검사하지 않습니다.

## 안전

기준일과 출처를 밝히고 매수·매도 지시, 목표가 보장이나 수익 단정을 하지 않습니다. 인증 정보 접근, 자동 거래와 계정 조작을 수행하지 않습니다.
