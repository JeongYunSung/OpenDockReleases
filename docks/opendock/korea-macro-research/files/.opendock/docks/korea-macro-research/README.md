# Korea Macro Research 사용 안내

한국 거시경제 지표를 공식 출처, 기준일과 시나리오로 조사합니다.

## 시작하기

1. 질문, 관심 지표, 기준일과 비교 기간을 알려줍니다.
2. agent에게 리서치를 직접 요청합니다: “최근 한국 물가와 금리 흐름을 공식 출처와 함께 정리해줘.”
3. 출처와 지표를 따로 정리할 필요가 있으면 `MACRO_RESEARCH_RUN.md`에서 필요한 section만 골라 사용합니다.

## 검토

평소에는 요청한 리서치를 바로 작성합니다. 사용자가 검토를 요청하면 현재 결과물만 `KOREA_MACRO_RESEARCH.md` 기준으로 AI가 직접 검토하며, 과거 결과물이나 프로젝트 전체는 검사하지 않습니다.

사용자가 만든 결과물은 사용자 소유이며 Dock 업데이트나 제거 대상이 아닙니다.
