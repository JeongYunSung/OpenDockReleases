# Agent Security 사용 안내

현재 변경 범위의 보안 위험, 영향, 근거와 완화책을 검토합니다.

## 바로 사용하기

1. `.opendock/docks/agent-security/SECURITY.md`에서 보안 검토 기준을 확인합니다.
2. AI에게 “이번 인증 변경의 위협, 영향, 완화책을 검토해줘”처럼 결과를 직접 요청합니다.
3. 정형화된 리뷰가 필요할 때만 `SECURITY_REVIEW.md`를 열고 현재 변경과 관련된 섹션만 사용합니다.

## 검토 요청

사용자가 검토를 요청하면 AI가 현재 결과물만 `SECURITY.md` 기준으로 직접 살펴보고 민감 영역, 신뢰 경계, 공격 가능성, 영향, 근거, 완화책과 잔여 위험을 설명합니다. 명시적 요청이 없으면 과거 작업이나 프로젝트 전체로 범위를 넓히지 않습니다.

사용자가 만든 결과물은 사용자 소유이며, Dock이 설치한 파일이 아니라면 OpenDock 업데이트나 제거가 삭제하지 않습니다.
