# Customer Support

Support playbooks, FAQ structure, ticket triage, escalation, and reusable response prompts.

## Outcome

Customer-facing answers become clearer, policy-aware, and easier to escalate without losing empathy.

## Best For

Support leads, CX teams, founders, and operations teams handling customer issues.

## What Gets Installed

- Shared agent instructions: `AGENTS.md`, `CLAUDE.md`, and `GEMINI.md`
- Reusable skills for OpenDock/OMA, Codex, and Claude Code
- A Cursor project rule scoped to this dock
- Role-specific reference files and prompt templates

## First Run

`opendock install opendock/customer-support@1.0.0`

After installation, open `README.md` in the target project and follow the first three steps.
