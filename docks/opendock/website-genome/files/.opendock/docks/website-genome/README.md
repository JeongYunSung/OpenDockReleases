# Website Genome 사용 안내
공개 웹사이트의 타이포그래피, 색상 역할, 레이아웃과 상호작용 원리를 분석합니다.
## 시작하기
1. `.opendock/docks/website-genome/WEBSITE_GENOME_PLAYBOOK.md`에서 필요한 입력과 결과 기준을 확인합니다.
2. agent에게 다음처럼 요청합니다: “이 공개 사이트의 디자인 체계를 desktop과 mobile 기준으로 분석해줘.”
3. 작업 메모가 필요하면 `.opendock/templates/website-genome/RUN.md`에서 관련 섹션만 선택해 사용합니다.
## 작업과 검토
평소에는 별도 준비 없이 바로 작업합니다. 사용자가 검토를 요청하면 AI가 현재 결과물만 `WEBSITE_GENOME_PLAYBOOK.md` 기준으로 직접 검토합니다.
사용자가 만든 결과물은 사용자 소유이며, Dock이 설치한 파일이 아니라면 OpenDock 업데이트나 제거가 삭제하지 않습니다.
