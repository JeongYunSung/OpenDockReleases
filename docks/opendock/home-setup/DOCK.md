# Home Setup

집의 실측, 보유 물품, 기능 구역과 예산을 근거로 구매·배치·설치 순서를 정리합니다.

## 설치 후 생기는 것

- `.opendock/docks/home-setup/README.md`: 빠른 사용 안내
- `.opendock/docks/home-setup/HOME_SETUP_PLAYBOOK.md`: 집 설정 계획 기준
- `.opendock/templates/home-setup/RUN.md`: 선택형 작업 메모
- `.agents/skills/opendock-home-setup/SKILL.md`: agent 작업 절차
- 루트 `AGENTS.md`: 짧은 요청 라우팅과 안전 규칙

## 사용 방법

방과 출입구 치수, 보유 물품, 예산, 생활 방식과 안전 제약을 알려주면 배치와 구매 순서를 바로 정리합니다. 템플릿은 정리가 필요할 때만 사용하고 현재 작업에 필요한 section만 선택합니다.

> 거실 치수와 보유 가구를 줄게. 통로 폭을 확인해 배치와 구매 순서를 정리해줘.

## 검토 방법

사용자가 검토를 요청하면 현재 결과물만 `HOME_SETUP_PLAYBOOK.md` 기준으로 AI가 직접 검토합니다. 요청하지 않은 과거 결과물이나 프로젝트 전체는 검사하지 않습니다.

## 안전

상세 주소, 출입 코드, Wi-Fi 비밀번호와 장기 부재 일정을 기록하지 않습니다. 전기·가스·벽 고정 작업은 지역 규정, 제품 설명서와 전문가 확인을 우선합니다.
