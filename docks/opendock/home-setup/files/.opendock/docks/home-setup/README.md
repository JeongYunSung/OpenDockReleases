# Home Setup 사용 안내

집의 실측, 보유 물품, 기능 구역과 예산을 근거로 구매·배치·설치 순서를 정리합니다.

## 시작하기

1. 방과 출입구 치수, 보유 물품, 예산, 생활 방식과 안전 제약을 알려줍니다.
2. agent에게 계획을 직접 요청합니다: “통로 폭을 확인해 방별 배치와 구매 순서를 정리해줘.”
3. 여러 치수와 결정을 따로 정리할 필요가 있으면 `RUN.md`에서 필요한 section만 골라 사용합니다.

## 검토

평소에는 요청한 결과물을 바로 작성합니다. 사용자가 검토를 요청하면 현재 결과물만 `HOME_SETUP_PLAYBOOK.md` 기준으로 AI가 직접 검토하며, 과거 결과물이나 프로젝트 전체는 검사하지 않습니다.

결과 파일은 사용자 소유이며 Dock 업데이트나 제거 대상이 아닙니다.

## 주의

상세 주소, 출입 코드, Wi-Fi 비밀번호와 장기 부재 일정을 제공하지 않습니다. 바뀔 수 있는 가격·제품·규정 정보는 출처와 확인 날짜를 남기고 실제 행동 전에 다시 확인합니다.
