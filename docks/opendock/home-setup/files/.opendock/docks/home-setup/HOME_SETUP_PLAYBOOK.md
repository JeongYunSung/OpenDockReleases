# Home Setup Playbook

## 1. 계획 입력

가구 구성, 생활 방식, 선호 스타일, 임대·접근성·안전 제약, 방과 반입 경로 치수, 보유 가구와 예산을 사실·가정·확인 대기로 분리합니다. 공개 가능한 계획에는 상세 주소, 방범 설비 위치, 출입 코드, Wi-Fi 비밀번호와 장기 부재 일정을 넣지 않습니다.

## 2. 측정 기준

- 모든 값에 `mm`, `cm`, `m`, `in`, `ft` 중 단위를 붙입니다.
- 벽 길이뿐 아니라 문 열림, 몰딩, 콘센트, 창, 난방·환기와 고정 설비 위치를 기록합니다.
- 반입에는 현관·복도·계단 회전부·엘리베이터·목적 방 문의 최소 유효 치수를 사용합니다.
- 가구 외형 치수에 포장, 손잡이, 조립 공간과 작업 여유를 더합니다.
- 실측하지 않은 값은 가정과 허용 오차를 표시하고 구매 전 재측정합니다.

## 3. 기능 구역과 우선순위

각 방을 수면, 업무, 식사, 수납, 휴식, 운동과 돌봄 같은 실제 행동 단위로 나눕니다. 안전, 필수 생활, 반복 동선, 편의와 스타일 순서를 기본으로 조정합니다.

## 4. 재고와 예산

보유 품목은 크기, 상태, 목적 방과 유지·수리·처분 결정을 기록합니다. 필요 품목은 해결할 기능, 필수 치수 범위, 예산 상한, 선행 측정과 대체 수단을 둡니다. 예산은 방과 범주별로 나누고 배송·설치와 예비비를 포함합니다.

## 5. 구매와 설치 순서

측정·안전 확인, 필수 기반 설치, 대형 가구, 수납, 조명·network, 장식 순으로 의존성을 정리합니다. 후보마다 외형 치수, clearance, 사용 공간, 전원과 발열, 배송·조립 조건과 반품 비용을 비교합니다.

## 6. 전원, network와 안전

- 멀티탭과 연장선을 상시 배선 대체로 가정하지 않습니다.
- 고부하 기기, 접지, 습식 공간, 가스와 벽체 내부 작업은 설명서·건물 규정과 자격 요건을 확인합니다.
- network 계획에는 장비 위치와 유선 경로만 기록하고 SSID·비밀번호·공인 IP는 남기지 않습니다.
- 피난로, 전도 방지, 코드와 아동·반려동물 위험을 확인합니다.

## 7. 보류와 결정 기록

치수, 예산, 사용자 합의, 제품 사양 또는 임대인 승인이 부족한 항목은 보류합니다. 보류에는 부족한 근거, 해제 조건과 재확인 시점을 기록합니다. 결정에는 고려한 대안, 사실과 가정, 예산 영향과 되돌림 조건을 남깁니다.

## 8. 선택 가능한 산출물

현재 요청에 필요한 항목만 선택합니다.

- 공간·반입 경로 치수와 기능 구역
- 보유·필요 품목과 예산
- 배치·clearance 검토
- 구매·설치 순서와 보류 항목
- 전원·network·안전 확인
- 결정 근거와 개인정보 처리

## 9. 검토 기준

사용자가 검토를 요청하면 AI는 현재 결과물만 치수 단위, 반입과 사용 clearance, 예산, 안전, 보류 조건, 출처와 개인정보 기준으로 직접 확인합니다. 현장 적합성과 안전은 설치 전 최종 실측과 사용자·전문가 확인이 필요합니다.
