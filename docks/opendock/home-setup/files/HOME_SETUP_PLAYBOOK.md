# Home Setup Playbook

## 1. 계획 입력

다음 정보를 사실, 가정, 확인 대기로 분리합니다.

- 가구 구성, 재택근무·수면·식사·취미·돌봄 방식
- 선호 스타일과 피해야 할 재료·색·관리 방식
- 임대 계약, 벽체 고정, 접근성, 아동·반려동물, 피난 제약
- 방별 폭·길이·높이, 문·복도·계단·엘리베이터의 유효 치수
- 보유 가구의 폭·깊이·높이, 상태, 유지 여부
- 전체 예산과 방별·범주별 상한

공개 가능한 계획에는 상세 주소와 동·호수, 방범 설비 위치, 출입 코드, Wi-Fi 비밀번호, 장기 부재 일정을 넣지 않습니다. 방은 `거실`, `침실 A`처럼 익명화합니다.

## 2. 측정 기준

- 모든 값에 `mm`, `cm`, `m`, `in`, `ft` 중 단위를 붙입니다.
- 벽 길이뿐 아니라 몰딩, 문 열림, 콘센트, 창, 난방·환기, 고정 설비 위치를 기록합니다.
- 반입에는 현관·복도·계단 회전부·엘리베이터·목적 방 문의 최소 유효 치수를 사용합니다.
- 가구 외형 치수에 포장 두께, 손잡이, 조립 공간과 작업 여유를 더합니다.
- 실측하지 않은 값으로 구매나 배치를 권고하지 않습니다. 임시 가정이면 허용 오차와 재측정 기한을 둡니다.

## 3. 기능 구역과 우선순위

각 방을 수면, 업무, 식사, 수납, 휴식, 운동, 돌봄처럼 실제 행동 단위로 나눕니다. 구역마다 사용자, 빈도, 필요한 면적·전원·조명·네트워크, 충돌 동선을 기록합니다.

우선순위는 다음 순서를 기본으로 조정합니다.

1. 피난, 전기, 가스, 전도, 아동·반려동물 안전
2. 수면, 위생, 식사, 필수 업무
3. 수납과 반복 동선 개선
4. 편의와 스타일 완성

## 4. 재고와 예산

보유 품목은 크기, 상태, 목적 방, 유지·수리·처분 결정을 기록합니다. 필요 품목은 해결할 기능, 필수 치수 범위, 예산 상한, 선행 측정, 대체 수단을 둡니다.

예산은 방과 범주를 동시에 볼 수 있어야 합니다. 가구, 조명, 수납, 네트워크, 안전, 설치·배송, 예비비를 분리하고 확정액과 추정액을 구분합니다. 추천에는 사용한 예산 가정을 직접 연결합니다.

## 5. 구매와 설치 순서

측정·안전 확인, 필수 기반 설치, 대형 가구, 수납, 조명·네트워크, 장식 순으로 의존성을 정리합니다. 반입이나 사용성 검증이 끝나기 전에 같은 공간의 후속 구매를 확정하지 않습니다.

각 후보는 외형 치수, 필요한 clearance, 문·서랍·의자 사용 공간, 전원과 발열, 배송·조립 조건, 반품 비용을 비교합니다. 정확한 제품 정보를 사용하면 제조사 HTTPS URL과 조회일을 남깁니다.

## 6. 전원, 네트워크, 안전

- 멀티탭과 연장선은 상시 배선 대체로 가정하지 않습니다.
- 고부하 기기, 접지, 습식 공간, 가스, 벽체 내부 작업은 제품 설명서·건물 규정과 자격 요건을 확인합니다.
- Wi-Fi 계획에는 공유기 위치, 유선 경로, 음영 구역만 기록하고 SSID·비밀번호·공인 IP는 남기지 않습니다.
- 피난로, 전도 방지, 모서리, 코드, 블라인드 줄, 세제·약 보관 등 가구별 위험을 확인합니다.

## 7. 보류와 결정 기록

치수, 예산, 사용자 합의, 제품 사양, 임대인 승인이 부족한 항목은 보류 목록으로 이동합니다. 보류에는 부족한 근거, 책임자, 해제 조건, 재검토일을 기록합니다.

결정 기록에는 날짜, 결정, 고려한 대안, 사용한 사실과 가정, 예산 영향, 되돌림 조건을 남깁니다. 하네스 통과는 현장 안전이나 제품 적합성의 보장이 아니므로 설치 전 최종 실측과 사용자 승인을 별도로 수행합니다.
