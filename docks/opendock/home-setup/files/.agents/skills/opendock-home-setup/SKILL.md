# OpenDock Home Setup

## 작업 순서

1. `.opendock/docks/home-setup/HOME_SETUP_PLAYBOOK.md`를 읽고 가구 구성, 생활 방식, 임대·접근성·안전 제약을 확인합니다.
2. 방·출입구·가구 치수, 보유·필요 품목, 기능 구역과 예산을 사실·가정·확인 대기로 구분합니다.
3. 모든 추천에 치수와 예산 근거를 연결하고 근거가 부족한 항목은 보류합니다.
4. 제품·안전·시점 의존 정보에는 출처와 확인일을 남깁니다.
5. 요청된 배치, 구매 순서와 설치 계획을 바로 작성합니다.
6. 작업 메모가 유용하면 선택 템플릿에서 필요한 section만 사용합니다.
7. 사용자가 검토를 요청하면 현재 결과물만 playbook 기준으로 AI가 직접 검토합니다.

## 결과 기준

- 치수에는 단위, 예산에는 통화와 범위를 둡니다.
- fit과 clearance는 반입 경로와 실제 사용 동작을 포함합니다.
- 구매 순서, 보류 조건과 결정 이유를 추적할 수 있어야 합니다.

## 안전

상세 주소, 출입·Wi-Fi credential, 방범 위치와 민감한 생활 정보를 최소화합니다. 구매, 결제, 공사, 예약과 파괴 명령은 자동 실행하지 않습니다.
