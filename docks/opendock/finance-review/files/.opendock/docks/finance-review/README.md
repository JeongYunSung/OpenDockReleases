# Finance Review 사용 안내

수입, 지출, 반복 결제, 목표와 예산 차이를 개인정보 없는 집계로 검토합니다.

## 시작하기

1. 검토할 기간, 기준 통화, 수입·지출 합계, 예산, 반복 결제와 목표를 알려줍니다.
2. agent에게 보고서를 직접 요청합니다: “계좌번호는 빼고 카테고리별 차이와 다음 달 조정안을 정리해줘.”
3. 작업 내용을 따로 정리할 필요가 있으면 `RUN.md`에서 필요한 section만 골라 사용합니다.

## 검토

평소에는 요청한 결과물을 바로 작성합니다. 사용자가 검토를 요청하면 현재 결과물만 `FINANCE_REVIEW_PLAYBOOK.md` 기준으로 AI가 직접 검토하며, 과거 결과물이나 프로젝트 전체는 검사하지 않습니다.

결과 파일은 사용자 소유이며 Dock 업데이트나 제거 대상이 아닙니다.

## 주의

계좌·카드 번호, credential, 원본 명세서와 전체 거래 메모는 제공하지 않습니다. 바뀔 수 있는 정보는 출처와 확인 날짜를 남기고 실제 행동 전에 다시 확인합니다.
