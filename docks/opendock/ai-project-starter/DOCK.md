# AI Project Starter

프로젝트 근거를 바탕으로 AI 협업 맥락, 역할, 규칙, 작업 절차, 품질 기준과 onboarding을 `.ai/`에 정리합니다.

## 설치 후 생기는 것

- `.opendock/docks/ai-project-starter/README.md`: 사용 안내
- `.opendock/docks/ai-project-starter/AI_PROJECT_STARTER_PLAYBOOK.md`: 프로젝트 구성 가이드
- `.opendock/templates/ai-project-starter/RUN.md`: 선택형 작업 메모 템플릿
- `.agents/skills/opendock-ai-project-starter/SKILL.md`: AI 작업 절차
- 루트 `AGENTS.md`: 짧은 요청 라우팅과 안전 규칙

## 사용 방법

평소에는 플레이북 기준으로 요청을 바로 처리합니다. 템플릿은 선택 사항이며 프로젝트에 필요한 섹션만 골라 사용합니다.

> 이 저장소 근거로 `.ai/PROJECT.md`의 목표, 규칙, 도구와 onboarding을 만들어줘.

사용자가 검토를 요청하면 AI가 현재 `.ai/` 결과물만 플레이북 기준으로 직접 검토하고 근거, 책임 경계, 안전 규칙, 작업 절차와 onboarding의 공백을 설명합니다.

## 안전

기존 vendor 설정을 직접 덮어쓰거나 secret과 불필요한 개인정보를 수집하지 않습니다. 승인 없이 관련 없는 파일 변경, 배포, 이전, package 설치 또는 파괴적 명령을 실행하지 않습니다.
