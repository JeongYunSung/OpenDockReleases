# Content Creator 사용 안내

출처, 독자, 채널과 권리를 고려해 콘텐츠 초안을 만듭니다.

## 시작하기

1. `CONTENT_CREATOR.md`에서 필요한 입력과 채널별 기준을 확인합니다.
2. agent에게 결과물을 직접 요청합니다: “이 자료를 근거로 실무 독자용 블로그 초안을 작성해줘.”
3. 작업 내용을 따로 정리할 필요가 있으면 `CONTENT_RUN.md`에서 필요한 section만 골라 사용합니다.

## 검토

평소에는 요청한 결과물을 바로 작성하거나 수정합니다. 사용자가 검토를 요청하면 현재 결과물만 `CONTENT_CREATOR.md` 기준으로 AI가 직접 검토하며, 과거 결과물이나 프로젝트 전체는 검사하지 않습니다.

사용자가 만든 결과물은 사용자 소유이며 Dock 업데이트나 제거 대상이 아닙니다.
