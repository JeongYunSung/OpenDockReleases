# Content Creator

출처, 독자, 채널과 권리를 고려해 콘텐츠 초안을 만듭니다.

## 설치 후 생기는 것

- `.opendock/docks/content-creator/README.md`: 빠른 사용 안내
- `.opendock/docks/content-creator/CONTENT_CREATOR.md`: 콘텐츠 제작 기준
- `.opendock/templates/content-creator/CONTENT_RUN.md`: 선택형 작업 메모
- `.agents/skills/opendock-content-creator/SKILL.md`: agent 작업 절차
- 루트 `AGENTS.md`: 짧은 요청 라우팅과 안전 규칙

## 사용 방법

평소에는 콘텐츠 제작 기준에 따라 요청된 결과물을 바로 작성하거나 수정합니다. 템플릿은 정리가 필요할 때만 사용하고 현재 작업에 필요한 section만 선택합니다.

> 이 자료를 근거로 실무 독자용 블로그 초안을 작성해줘.

## 검토 방법

사용자가 검토를 요청하면 현재 결과물만 `CONTENT_CREATOR.md` 기준으로 AI가 직접 검토합니다. 요청하지 않은 과거 결과물이나 프로젝트 전체는 검사하지 않습니다.

## 안전

출처와 사용 권리를 확인하고 비밀이나 불필요한 개인정보를 콘텐츠에 넣지 않습니다. 승인 없이 관련 없는 파일 수정·삭제, credential 접근, 배포, 이전 작업 또는 위험한 명령을 실행하지 않습니다.
