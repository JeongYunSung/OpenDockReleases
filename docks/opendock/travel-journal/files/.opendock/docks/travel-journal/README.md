# Travel Journal 사용 안내

## 먼저 이렇게 요청하세요

- 알려줄 것: 사진·메모의 안전한 이름, 여행 날짜, 장소 범위, 함께한 사람의 공개 동의
- 받을 결과: `travel-journal/` 아래에 시간순 기록, 짧은 캡션, 긴 이야기와 공개용 가림 확인표
- 예시: "교토 여행 사진 목록과 메모를 줄게. 동행자는 가명으로 쓰고 짧은 글과 긴 글을 travel-journal/kyoto.md로 정리해줘."

## 사용 순서

1. 필요한 정보를 알려주고 빼야 할 개인정보도 함께 말합니다.
2. 확인된 날짜·장소와 기억이 불확실한 내용을 구분해 이야기와 캡션을 만듭니다.
3. 결과는 사용자 소유 `travel-journal/` 아래에 저장합니다.
4. 작업 메모가 필요하면 선택 양식에서 관련 섹션만 골라 사용합니다.

평소에는 별도 준비 없이 바로 작업합니다. 사용자가 검토를 요청하면 AI가 현재 결과물만 `.opendock/docks/travel-journal/TRAVEL_JOURNAL_PLAYBOOK.md` 기준으로 직접 검토합니다.

## 주의할 점

- 동의 없는 실명, 얼굴 설명, 연락처와 신원 단서는 공개하지 않습니다.
- 정확한 GPS, 숙소·자택 위치와 원본 EXIF는 넣지 않습니다.
- 바뀔 수 있는 정보는 링크와 확인 날짜를 남기고 실제 행동 전에 다시 확인합니다.
- `travel-journal/` 아래의 결과는 사용자 소유이며 Dock 업데이트나 제거 대상이 아닙니다.
