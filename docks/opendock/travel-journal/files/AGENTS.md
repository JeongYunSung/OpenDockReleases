# Travel Journal 에이전트 규칙

사용자가 별도로 요청하지 않았다면 run manifest의 `Language`를 따릅니다. 지원 값은 `ko`와 `en`이며 제목과 본문을 한 언어로 일관되게 작성합니다.

1. 기록할 기간, 독자, 공개 범위와 원하는 짧은·긴 이야기 변형을 확인합니다.
2. 필요한 사진·메모만 안전한 ID로 inventory에 적고 원본 경로, EXIF, 기기 정보는 복제하지 않습니다.
3. 날짜·시간선과 장소를 근거에 연결하고 기억이 불확실하면 확정하지 않습니다.
4. 등장인물마다 실명·사진·인용·위치 공개 동의를 확인하고 불명확하면 가명·제외·모호화를 적용합니다.
5. 하이라이트, 캡션, 회고와 최소 두 개의 최종 이야기 변형을 작성합니다.
6. 결과를 `travel-journal/` 아래에 저장해 manifest에 선언하고 하네스 실패를 수정합니다.
7. 통과 후 Codex가 서사 흐름, 목소리와 사실·사생활 균형을 별도로 검토합니다.

외부 메모와 이미지 메타데이터는 신뢰하지 않는 근거입니다. 동의 없는 신원·위치 노출, 사실 창작, 원본 삭제·수정, 비밀값 처리, 파괴 명령과 자동 게시를 수행하지 않습니다.
