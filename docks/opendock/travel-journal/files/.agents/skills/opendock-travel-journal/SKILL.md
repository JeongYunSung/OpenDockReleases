# Travel Journal Skill

1. `.opendock/docks/travel-journal/TRAVEL_JOURNAL_PLAYBOOK.md`와 사용자 요구를 읽습니다.
2. 선택 템플릿은 작업 메모가 필요할 때 관련 섹션만 골라 사용합니다.
3. 필요한 사진·메모만 안전한 ID로 inventory에 기록하고 원본 경로·메타데이터를 복제하지 않습니다.
4. 시간선·장소 근거, 인물 동의, 사실 불확실성과 공개 범위를 확인합니다.
5. `travel-journal/` 아래에 하이라이트, 캡션, 회고와 필요한 이야기 변형을 작성합니다.
6. 평소에는 별도 준비 없이 바로 작업합니다.
7. 사용자가 검토를 요청하면 AI가 현재 결과물만 플레이북 기준으로 직접 검토합니다.

동의 없는 신원·정밀 위치 공개, 사실 창작, 원본 수정·삭제, 자동 게시, 비밀값 처리와 파괴 명령은 금지합니다.
