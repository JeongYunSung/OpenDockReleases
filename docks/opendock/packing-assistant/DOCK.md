# Packing Assistant

## 무엇을 알려주면 무엇이 나오나요

- 알려줄 것: 목적지, 날짜, 날씨, 활동, 숙소 세탁 여부, 이동 수단과 수하물 제한
- 정리되는 것: `packing/` 아래에 수량이 있는 짐 목록, 입기·세탁 계획, 현지 구매와 출발 직전 확인표
- 요청 예시: "12월 삿포로 5박과 기내 7kg 조건을 줄게. 수량과 세탁 계획이 있는 목록을 packing/list.md로 정리해줘."

## 설치되는 안내

- `.opendock/docks/packing-assistant/README.md`: 가장 쉬운 사용 순서
- `.opendock/docks/packing-assistant/PACKING_PLAYBOOK.md`: 자세한 작성 기준
- `.opendock/templates/packing-assistant/RUN.md`: 필요한 항목만 골라 쓰는 선택 템플릿
- `.agents/skills/opendock-packing-assistant/SKILL.md`: agent 작업 절차

## 사용 순서

1. 필요한 정보를 받고 민감한 값은 먼저 뺍니다.
2. 날씨와 활동을 기준으로 수량을 계산하고 다시 입기·세탁·현지 구매 항목을 나눕니다.
3. 템플릿은 필요한 섹션만 선택해 사용합니다.
4. 결과를 사용자 소유 `packing/` 아래에 바로 저장합니다.

## 검토

사용자가 검토를 요청하면 AI가 현재 결과물만 짐싸기 플레이북 기준으로 직접 검토하고, 수량·날씨·활동·수하물·세탁·출발 직전 확인 문제를 수정합니다.

## 안전

- 약의 복용량, 중단 또는 대체를 지시하지 않습니다.
- 여권 번호, 예약 코드와 자세한 건강 정보는 목록에 적지 않습니다.
- 웹의 가격·규정·운영 정보는 출처와 확인 날짜를 남기고 중요한 결정 전에 다시 확인합니다.
- 사용자가 만든 `packing/` 파일은 Dock을 업데이트하거나 제거해도 삭제하지 않습니다.
