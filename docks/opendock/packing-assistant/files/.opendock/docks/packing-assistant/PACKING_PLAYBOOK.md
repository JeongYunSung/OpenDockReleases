# 짐싸기 플레이북

## 입력

목적지, 출발·귀국 날짜, 예상 기온·강수·바람, 활동, 숙소 난방·세탁, 도보·차량·항공 이동, 기내·위탁 수하물 제한과 직접 제공된 의료 제약을 확인합니다. 여권 번호, 예약 코드, 진단명과 처방 식별자는 기록하지 않습니다.

## 수량 산정

여행 일수만큼 단순 복제하지 않습니다. 착용 빈도, 오염 가능성, 레이어링, 세탁 날짜, 건조 시간과 예비 1회를 기준으로 수량을 설명합니다. 각 항목은 체크박스, 수량, 기내·위탁·착용 위치를 갖습니다.

## 상황별 카테고리

여행 방식과 활동에 해당하는 카테고리만 선택합니다.

- 의류·신발: 날씨와 활동별 레이어, 재착용과 세탁 연결
- 위생·생활: 용량 제한과 현지 구매 가능 여부
- 서류·결제: 문서 종류만 기록하고 번호나 코드를 노출하지 않음
- 전자기기: 전압·플러그·배터리 운송 규정 확인
- 의약품: 기존 처방과 원래 용기, 의료진·약사 및 공식 반입 규정 확인

## 마지막 점검

출발 당일 넣을 휴대전화, 지갑, 열쇠, 충전 중인 기기, 냉장 보관 항목과 문서를 별도 체크리스트로 둡니다. 잊어도 현지에서 안전하게 구매할 수 있는 항목과 구매하기 어려운 필수품을 구분합니다.

## 의료 경계

약의 종류, 용량, 복용 시각을 새로 정하거나 중단·대체하지 않습니다. 의약품은 사용자가 가진 처방과 전문가 지시를 따르고 항공사·세관·보건기관의 최신 운반 조건을 확인하도록 안내합니다.

## 검토 기준

포함한 항목에는 필요한 수량이 있고 날씨·활동·수하물 조건과 연결되어야 합니다. 해당되는 경우 재착용·세탁, 출발 직전, 현지 구매와 재확인 주의사항을 포함하고 개인정보를 가립니다.

평소에는 요청한 짐 목록을 바로 작성하고, 선택 템플릿에서는 관련 섹션만 사용합니다. 사용자가 검토를 요청하면 AI가 현재 결과물만 이 playbook 기준으로 직접 확인하고 수량, 조건 연결, 주의사항과 개인정보 문제를 수정합니다.
