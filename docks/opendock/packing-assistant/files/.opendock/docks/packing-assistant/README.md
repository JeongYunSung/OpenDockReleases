# Packing Assistant 사용 안내

## 먼저 이렇게 요청하세요

- 알려줄 것: 목적지, 날짜, 날씨, 활동, 숙소 세탁 여부, 이동 수단과 수하물 제한
- 받을 결과: `packing/` 아래에 수량이 있는 짐 목록, 입기·세탁 계획, 현지 구매와 출발 직전 확인표
- 예시: "12월 삿포로 5박과 기내 7kg 조건을 줄게. 수량과 세탁 계획이 있는 목록을 packing/list.md로 정리해줘."

## 사용 순서

1. 필요한 정보를 알려주고 빼야 할 개인정보도 함께 말합니다.
2. 날씨와 활동을 기준으로 수량을 계산하고 다시 입기·세탁·현지 구매 항목을 나눕니다.
3. 결과는 사용자 소유 `packing/` 아래에 바로 저장합니다.
4. 필요하면 `.opendock/templates/packing-assistant/RUN.md`에서 관련 섹션만 골라 사용합니다.

## 검토

검토를 요청하면 AI가 현재 결과물만 `.opendock/docks/packing-assistant/PACKING_PLAYBOOK.md` 기준으로 직접 확인합니다. 수량 산정, 날씨·활동 연결, 수하물 제약, 재착용·세탁, 출발 직전과 현지 구매 항목을 점검하고 필요한 내용을 수정합니다.

## 주의할 점

- 약의 복용량, 중단 또는 대체를 지시하지 않습니다.
- 여권 번호, 예약 코드와 자세한 건강 정보는 목록에 적지 않습니다.
- 바뀔 수 있는 정보는 링크와 확인 날짜를 남기고 실제 행동 전에 다시 확인합니다.
- `packing/` 아래의 결과는 사용자 소유이며 Dock 업데이트나 제거 대상이 아닙니다.
