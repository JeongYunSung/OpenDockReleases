# Packing Assistant 품질 게이트

## 입력과 근거

템플릿을 run manifest로 복사합니다. 목적지, 날짜, 최신 기상, 활동, 숙소·교통, 수하물과 최소 의료 제약을 기록하고 공식 출처 URL·접근일을 남깁니다.

## 목록 설계

수량, 체크박스와 운반 위치가 있는 카테고리별 목록을 작성합니다. 날씨 레이어, 재착용·세탁, 서류·전자·의약품 주의, 출발 직전 목록과 현지 구매 대안을 연결합니다.

## 검사

산출물을 `packing/` 아래에 저장해 선언하고 `node .opendock/harness/opendock__packing-assistant/check.mjs <manifest-path>`를 실행합니다. 실패 규칙을 수정해 반복합니다.

## 별도 검토

Codex와 사람이 실제 가방 무게, 최신 운송·반입 규정과 개인 적합성을 확인합니다. 하네스 통과나 외부 모델 응답은 이를 보장하지 않습니다.
