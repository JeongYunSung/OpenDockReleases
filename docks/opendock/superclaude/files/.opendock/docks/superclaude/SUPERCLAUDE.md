# SuperClaude 사용 가이드

## 사용 시점

- Claude Code command와 persona를 체계적으로 관리할 때
- 반복 workflow 구성을 확인하거나 설치할 때
- 기존 설정과 새 profile의 충돌을 검토할 때

## 변경 전

- 적용할 SuperClaude version과 command를 확인합니다.
- 바뀔 HOME 또는 project 설정 경로를 확인합니다.
- 기존 project rule과 충돌하는 지시가 있는지 검토합니다.

## 금지

- 승인 없이 Claude Code HOME이나 user config를 변경하지 않습니다.
- SuperClaude command를 project rule보다 우선하지 않습니다.
- account, token, secret을 설정이나 작업 기록에 저장하지 않습니다.
