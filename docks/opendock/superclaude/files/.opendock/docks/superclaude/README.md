# SuperClaude

`superclaude`를 사용해 Claude Code command, persona, workflow 구성을 확인하고 관리합니다.

## 빠른 확인

```sh
superclaude --version
superclaude --help
opendock doctor
```

`superclaude install`처럼 Claude Code 설정을 바꾸는 명령은 변경 위치와 영향을 확인하고 사용자 승인 후 실행합니다.

상세 원칙은 `SUPERCLAUDE.md`, 선택적 변경 기록 양식은 `SUPERCLAUDE_RUN.md`를 참고합니다.

이 도구 Dock에는 별도 정밀 검사 도구가 없습니다. 설치 검수는 `opendock doctor`와 실제 `superclaude` 명령으로 진행합니다.
