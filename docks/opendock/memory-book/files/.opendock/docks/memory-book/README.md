# Memory Book 사용 안내

## 먼저 이렇게 요청하세요

- 알려줄 것: 사진·메모·편지의 안전한 이름, 기간, 주제, 독자, 인물별 공개 동의와 공유 범위
- 받을 결과: `memories/` 아래에 출처가 표시된 시간순 이야기, 캡션, 회고와 가림 확인표
- 예시: "2019년 가족 여행 사진 목록과 메모를 줄게. 출처 번호를 붙이고 위치 정보는 가려 memories/book.md로 정리해줘."

## 사용 순서

1. 필요한 정보를 알려주고 빼야 할 개인정보도 함께 말합니다.
2. 자료마다 번호를 붙이고 확인된 사실과 기억이 불확실한 부분을 나눕니다.
3. 결과는 사용자 소유 `memories/` 아래에 바로 저장합니다.
4. 필요하면 `.opendock/templates/memory-book/RUN.md`에서 관련 섹션만 골라 사용합니다.

## 검토

검토를 요청하면 AI가 현재 결과물만 `.opendock/docks/memory-book/MEMORY_BOOK_PLAYBOOK.md` 기준으로 직접 확인합니다. 출처 연결, 사실과 회고의 분리, 인물 동의, 위치·EXIF·GPS 가림과 공개 범위를 점검하고 필요한 내용을 수정합니다.

## 주의할 점

- 자료에 없는 사건, 말과 감정을 사실처럼 만들지 않습니다.
- 동의 없는 실명, 정확한 주소, 숙소, GPS와 EXIF 정보는 공개하지 않습니다.
- 바뀔 수 있는 정보는 링크와 확인 날짜를 남기고 실제 행동 전에 다시 확인합니다.
- `memories/` 아래의 결과는 사용자 소유이며 Dock 업데이트나 제거 대상이 아닙니다.
