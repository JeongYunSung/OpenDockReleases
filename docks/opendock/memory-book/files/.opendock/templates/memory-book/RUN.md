# Memory Book 선택 템플릿

이야기의 목적과 공개 범위에 필요한 섹션만 골라 사용하고, 관련 없는 섹션은 삭제하세요.

## 기간과 테마

시작일·종료일 또는 연도 범위, 중심 테마, 독자와 공유 범위를 작성하세요.

## 출처 목록

- [S1] 유형, 날짜, 제공자 역할, 동의 상태, 사용 범위

## 타임라인

- YYYY-MM-DD: source로 확인한 사건 요약 [S1]

## 인물 동의와 개인정보

표시 이름 또는 역할, 동의 상태, 허용 매체, 공유 범위를 기록합니다.

## 장소와 안전한 세분성

도시·지역·국가 수준만 사용하고 정확한 주소, 숙소, GPS와 미래 동선을 제외한 근거를 기록합니다.

## 하이라이트

- source로 확인한 핵심 장면과 선택 이유 [S1]

## 캡션

- 보이는 사실과 안전한 날짜·장소를 담은 짧은 caption [S1]

## 회고

현재 작성자의 해석과 감정을 사실 서술과 분리해 기록합니다.

## 가림과 제외

인물, 식별자, 정확한 위치, EXIF/GPS, 민감 사건의 redaction과 제외 결정을 기록합니다.

## 최종 서사와 출력 계획

형식, 독자, section 순서, caption 스타일, 공유 범위, 최종 동의 확인 단계를 기록합니다.

## 검증과 불확실성

확인된 사실, 미확인·충돌 자료, 추정 제외, source 누락과 최종 확인 담당자를 기록합니다.

## 확인 메모

- 사실성 또는 동의 확인이 필요한 항목
- 공개 전 추가로 가리거나 제외할 항목
