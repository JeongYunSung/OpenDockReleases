# Portfolio Case Study

프로젝트 경험을 근거 있는 문제, 기여, 과정과 결과의 사례 연구로 만듭니다.

## 설치 후 생기는 것

- `.opendock/docks/portfolio-case-study/README.md`: 사용 안내
- `.opendock/docks/portfolio-case-study/PORTFOLIO_CASE_STUDY_PLAYBOOK.md`: 사례 연구 기준
- `.opendock/templates/portfolio-case-study/RUN.md`: 필요한 항목만 골라 쓰는 선택 템플릿
- `.agents/skills/opendock-portfolio-case-study/SKILL.md`: agent 작업 절차
- 루트 `AGENTS.md`: 짧은 요청 라우팅과 안전 규칙

## 사용

> 이 프로젝트 기록으로 포트폴리오 사례 연구 초안을 만들어줘.

별도 작업 기록을 먼저 만들 필요 없이 결과를 바로 작성합니다. 템플릿은 요청에 필요한 섹션만 선택해 사용합니다.

## 검토

사용자가 검토를 요청하면 AI가 현재 결과물만 Portfolio Case Study Playbook 기준으로 직접 검토하고, 주장 근거·기여 범위·성과 수치·한계·개인정보 문제를 수정합니다.

## 안전

고객·회사 비밀과 개인정보를 제거하고 성과 수치를 꾸미지 않습니다. 관련 없는 파일 수정·삭제, credential 접근, 공개·배포와 파괴적 작업을 승인 없이 실행하지 않습니다.
