# Korea Real Estate Research 사용 안내

한국 부동산 시장을 거래, 공급, 정책과 지역별 위험으로 조사합니다.

## 시작하기

1. `.opendock/docks/korea-real-estate-research/KOREA_REAL_ESTATE_RESEARCH.md`에서 필요한 입력과 결과 기준을 확인합니다.
2. agent에게 “서울 아파트 시장을 기준일, 실거래와 공급 지표로 조사해줘”처럼 결과를 바로 요청합니다.
3. 필요하면 `.opendock/templates/korea-real-estate-research/REAL_ESTATE_RESEARCH_RUN.md`에서 질문에 맞는 섹션만 골라 사용합니다.

## 검토

검토를 요청하면 AI가 현재 결과물만 리서치 가이드 기준으로 직접 확인합니다. 공식 출처, 조회일과 기준일, 지역·주택·거래 범위, 반대 시나리오, 데이터 한계와 비추천 표현을 점검하고 필요한 내용을 수정합니다.

사용자가 만든 결과물은 사용자 소유이며, Dock이 설치한 파일이 아니라면 OpenDock 업데이트나 제거가 삭제하지 않습니다.
