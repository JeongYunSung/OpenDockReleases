# Group Trip

- 공동 여행 계획은 `.opendock/docks/group-trip/GROUP_TRIP_PLAYBOOK.md`를 기준으로 처리합니다.
- 평소에는 요청된 합의안, 일정 대안과 비용 배분을 바로 작성합니다.
- `.opendock/templates/group-trip/RUN.md`는 선택 사항이며 필요한 section만 사용합니다.
- 사용자가 검토를 요청하면 현재 결과물만 playbook 기준으로 AI가 직접 검토합니다.
- 요청하지 않은 과거 결과물이나 프로젝트 전체는 검사하지 않습니다.

## 안전

- 구성원이 직접 밝히지 않은 건강, 장애, 종교와 같은 민감한 특성을 추정하지 않습니다.
- 연락처, 상세 주소, 예약·결제 번호는 결과에 적지 않습니다.
- 예약, 구매, 결제, 제출과 원본 삭제는 사용자 승인 없이 실행하지 않습니다.
