# Group Trip 사용 안내

구성원별 선호, 제약과 예산을 공정한 결정 방식과 비용 배분으로 연결합니다.

## 시작하기

1. 여행 날짜, 구성원별 예산·선호·제약, 접근성 요구와 결정할 항목을 알려줍니다.
2. agent에게 합의안을 직접 요청합니다: “각자의 조건을 비교하고 공정한 투표안과 비용 배분을 정리해줘.”
3. 여러 의견을 따로 정리할 필요가 있으면 `RUN.md`에서 필요한 section만 골라 사용합니다.

## 검토

평소에는 요청한 결과물을 바로 작성합니다. 사용자가 검토를 요청하면 현재 결과물만 `GROUP_TRIP_PLAYBOOK.md` 기준으로 AI가 직접 검토하며, 과거 결과물이나 프로젝트 전체는 검사하지 않습니다.

결과 파일은 사용자 소유이며 Dock 업데이트나 제거 대상이 아닙니다.

## 주의

민감한 특성을 추정하지 않고 연락처, 상세 주소, 예약 번호와 결제 정보를 가립니다. 바뀔 수 있는 가격·운영 정보는 출처와 확인 날짜를 남기고 실제 행동 전에 다시 확인합니다.
