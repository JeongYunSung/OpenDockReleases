# Creative Generation Ultrawork

Before final handoff:

1. Complete `GENERATION_BRIEF.md`.
2. Create or analyze the asset.
3. Update `OUTPUT_MANIFEST.md`.
4. Run `node .opendock/harness/check.mjs`.
5. Revise until the harness passes.

Do not claim generated work is finished without traceable output metadata.
