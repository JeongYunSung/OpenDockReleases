# Creative Generation Ultrawork

Use this workspace as a loop for generated creative assets.

1. Read `GENERATION_BRIEF.md`.
2. Produce the requested artifact.
3. Record the exact output paths and generation metadata in `OUTPUT_MANIFEST.md`.
4. Run `node .opendock/harness/check.mjs`.
5. Revise until the harness passes.

Never leave generated assets undocumented.
