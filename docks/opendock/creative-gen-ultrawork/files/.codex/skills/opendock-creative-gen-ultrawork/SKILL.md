# Creative Generation Ultrawork

Run creative work as a loop:

1. Brief
2. Generate or analyze
3. Record manifest
4. Run harness
5. Revise

Required command:

```bash
node .opendock/harness/check.mjs
```

Generated assets must have output paths, prompt, tool, model, date, rights, review notes, and revision history in `OUTPUT_MANIFEST.md`.
