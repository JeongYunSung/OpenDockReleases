# Designer AI Prompts

## Review This UI

Review this UI for usability, hierarchy, accessibility, consistency, and
conversion risk. Lead with the highest-impact problems and give concrete fixes.

## Generate UX Audit

Create a UX audit with sections for user goal, friction points, information
architecture, accessibility, visual hierarchy, and suggested experiments.

## Create User Flow

Create a user flow for the target user and goal. Include entry point, decision
points, success state, failure states, and data needed at each step.

## Improve Onboarding

Suggest onboarding improvements for a first-time user. Keep the flow short,
reduce cognitive load, and define one measurable activation event.
