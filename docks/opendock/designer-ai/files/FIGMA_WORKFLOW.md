# Figma Workflow

## Standard Flow

1. Research
2. Wireframe
3. High fidelity
4. Prototype
5. Handoff

## Research

- Capture user problem, target segment, and comparable products.

## Wireframe

- Validate structure before visual polish.

## High Fidelity

- Apply design tokens, responsive rules, and component states.

## Prototype

- Cover the main happy path and key failure states.

## Handoff

- Include component names, spacing notes, state coverage, and open questions.
