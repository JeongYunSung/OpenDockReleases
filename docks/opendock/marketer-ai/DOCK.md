# Marketer AI

Campaign briefs, content calendars, SEO briefs, and ad experiment planning.

## Outcome

AI-assisted marketing starts from audience, offer, channel, evidence, and measurable experiment design.

## Best For

Marketing teams, founders, content leads, and growth operators.

## What Gets Installed

- Shared agent instructions: `AGENTS.md`, `CLAUDE.md`, and `GEMINI.md`
- Reusable skills for OpenDock/OMA, Codex, and Claude Code
- A Cursor project rule scoped to this dock
- Role-specific reference files and prompt templates

## First Run

`opendock install opendock/marketer-ai@1.0.0`

After installation, open `README.md` in the target project and follow the first three steps.
