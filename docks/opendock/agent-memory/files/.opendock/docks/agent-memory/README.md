# Agent Memory 사용 안내

다음 작업에 필요한 결정, 근거와 미해결 맥락만 안전하게 기억으로 남깁니다.

## 바로 사용하기

1. `.opendock/docks/agent-memory/MEMORY.md`에서 작성 기준을 확인합니다.
2. AI에게 “이번 작업에서 다음 세션에 남길 결정과 미해결 항목만 정리해줘”처럼 결과를 직접 요청합니다.
3. 정형화된 작업 메모가 필요할 때만 `MEMORY_RUN.md`를 열고 필요한 섹션만 사용합니다.

## 검토 요청

사용자가 검토를 요청하면 AI가 현재 결과물만 `MEMORY.md` 기준으로 직접 살펴보고 사실과 추측의 구분, 근거, 최신성, 후속 확인과 민감 정보 포함 여부를 설명합니다. 명시적 요청이 없으면 과거 기록이나 프로젝트 전체로 범위를 넓히지 않습니다.

사용자가 만든 결과물은 사용자 소유이며, Dock이 설치한 파일이 아니라면 OpenDock 업데이트나 제거가 삭제하지 않습니다.
