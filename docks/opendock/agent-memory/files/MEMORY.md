# MEMORY.md

이 파일은 프로젝트에서 AI agent가 반복해서 참고해야 할 기억 기준입니다.

## 기록할 것

- 프로젝트 구조와 주요 모듈
- 중요한 의사결정과 이유
- 반복되는 검증 명령과 결과 해석
- 외부 도구를 사용할 때의 workspace-local 경로
- 현재 알려진 위험, 공백, stale 정보

## 기록하지 말 것

- API key, access token, refresh token, password, private key
- 개인 식별정보
- private prompt 전문
- 임시 로그 전문

## 기록 형식

```markdown
## YYYY-MM-DD 주제

- 사실:
- 추측:
- 근거:
- 다음 확인:
```
