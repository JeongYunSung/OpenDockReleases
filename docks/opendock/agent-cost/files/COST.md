# COST.md

AI agent 사용량과 비용 추정을 기록하는 기준입니다.

## 기록할 것

- Provider
- Model
- Agent 또는 tool
- Task
- Date
- Owner
- Reason
- Token, time, run count, cost estimate 중 가능한 값

## 기록하지 말 것

- API key
- 결제 정보
- invoice 원문
- 개인 계정 식별 정보

## 기준

비용은 정확한 회계보다 의사결정 가능한 추적성을 우선합니다. 값이 추정치라면 estimate라고 표시하고 근거를 남깁니다.
