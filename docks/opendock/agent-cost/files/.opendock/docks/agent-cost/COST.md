# 비용 기록 가이드

AI 도구 사용량과 비용 추정을 의사결정에 필요한 수준으로 기록합니다.

## 기록 후보

- Provider와 model
- Agent 또는 tool
- Task와 사용 이유
- Date와 owner
- Token, 시간, 실행 횟수, 비용 추정 중 확인 가능한 값
- 결과 또는 제공 가치

모든 항목을 채울 필요는 없습니다. 현재 작업의 비용과 판단 근거를 설명하는 데 필요한 항목만 선택합니다.

## 기록하지 말 것

- API key, access token, password
- 결제 정보와 invoice 원문
- 개인 계정 식별 정보
- 근거 없이 확정한 비용

## 추정 기준

정확한 회계값이 아니면 추정이라고 표시합니다. 사용량, 단가, 실행 시간 또는 실행 횟수처럼 재확인 가능한 근거를 함께 남깁니다.

## AI 검토 기준

사용자가 검토를 요청하면 AI는 현재 결과물에서 비용 추정 근거, 주요 비용 요인, 제공 가치, 조정 가능성과 민감 정보 노출만 직접 확인합니다.
