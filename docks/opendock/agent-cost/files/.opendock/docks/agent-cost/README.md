# Agent Cost 사용 안내

AI 도구 사용량과 비용 추정 근거를 작업별로 남깁니다.

## 바로 사용하기

1. `.opendock/docks/agent-cost/COST.md`에서 기록 기준을 확인합니다.
2. AI에게 “이번 AI 작업의 모델, 사용 이유, 예상 비용과 결과를 기록해줘”처럼 결과를 직접 요청합니다.
3. 정형화된 기록이 필요할 때만 `USAGE_LOG.md` 또는 `COST_REVIEW.md`를 열고 필요한 섹션만 사용합니다.

## 검토 요청

사용자가 검토를 요청하면 AI가 현재 결과물만 `COST.md` 기준으로 직접 살펴보고 추정 근거, 비용 요인, 제공 가치와 민감 정보 포함 여부를 설명합니다. 명시적 요청이 없으면 과거 기록이나 프로젝트 전체로 범위를 넓히지 않습니다.

사용자가 만든 결과물은 사용자 소유이며, Dock이 설치한 파일이 아니라면 OpenDock 업데이트나 제거가 삭제하지 않습니다.
