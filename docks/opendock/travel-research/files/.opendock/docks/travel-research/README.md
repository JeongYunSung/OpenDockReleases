# Travel Research 사용 안내

## 먼저 이렇게 요청하세요

- 알려줄 것: 목적지, 여행 날짜, 체류 기간, 비교할 질문, 예산과 중요한 제약
- 받을 결과: `travel-research/` 아래에 출처와 확인 날짜가 있는 지역·교통·비용·규정 비교 자료
- 예시: "11월 리스본 6박이야. 숙박 지역 3곳을 교통과 비용으로 비교해 travel-research/lisbon.md로 정리해줘."

## 사용 순서

1. 필요한 정보를 알려주고 빼야 할 개인정보도 함께 말합니다.
2. 공식 자료를 먼저 확인하고 바뀔 수 있는 정보에는 링크와 확인 날짜를 남깁니다.
3. 결과는 사용자 소유 `travel-research/` 아래에 저장합니다.
4. 작업 메모가 필요하면 선택 양식에서 관련 섹션만 골라 사용합니다.

평소에는 별도 준비 없이 바로 작업합니다. 사용자가 검토를 요청하면 AI가 현재 결과물만 `.opendock/docks/travel-research/TRAVEL_RESEARCH_PLAYBOOK.md` 기준으로 직접 검토합니다.

## 주의할 점

- 안전, 가격, 영업시간과 규정이 확실하다고 보장하지 않습니다.
- 상세 숙소 주소, 여권·예약 번호, 연락처와 실시간 위치는 적지 않습니다.
- 바뀔 수 있는 정보는 링크와 확인 날짜를 남기고 실제 행동 전에 다시 확인합니다.
- `travel-research/` 아래의 결과는 사용자 소유이며 Dock 업데이트나 제거 대상이 아닙니다.
