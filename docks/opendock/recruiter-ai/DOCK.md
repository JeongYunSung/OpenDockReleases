# Recruiter AI

Job description, interview guide, scorecard, and candidate outreach workspace.

## Outcome

Hiring materials become structured, criteria-based, and easier to review for fairness and role fit.

## Best For

Recruiters, hiring managers, founders, and people teams.

## What Gets Installed

- Shared agent instructions: `AGENTS.md`, `CLAUDE.md`, and `GEMINI.md`
- Reusable skills for OpenDock/OMA, Codex, and Claude Code
- A Cursor project rule scoped to this dock
- Role-specific reference files and prompt templates

## First Run

`opendock install opendock/recruiter-ai@1.0.0`

After installation, open `README.md` in the target project and follow the first three steps.

## Pair With

- opendock/docs-ai
- opendock/customer-support

## Edition

This is the simple edition of `opendock/recruiter-ai`. It keeps setup small and installs the core workspace context. For a stronger specialist team, use the pro addon: [opendock/recruiter-ai-pro](https://hub.opendock.app/docks/opendock/recruiter-ai-pro).
